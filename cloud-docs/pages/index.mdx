# SciPhi: Effortless Deployment of Production-Ready RAG Pipelines

SciPhi is a cloud platform that empowers developers to effortlessly deploy and manage production-ready Retrieval-Augmented Generation (RAG) pipelines. Built on top of the robust R2R framework, SciPhi streamlines the process of setting up, deploying, and scaling RAG systems, enabling you to focus on building innovative applications.

## Getting Started

To get started with SciPhi, follow these simple steps:

1. **Link Your GitHub Repository**: [Connect your GitHub repository](/getting-started/deploying-your-first-pipeline) containing your RAG pipeline code to SciPhi, allowing seamless integration and deployment.

2. **Customize Your Configuration**: [Tailor your RAG pipeline](/getting-started/customize-your-config) to your specific requirements by modifying the `config.json` file, specifying vector database providers, embedding settings, language models, and more.

3. **Deploy with Ease**: [Deploy your RAG pipeline](/getting-started/deploying-your-first-pipeline) to the cloud with just a few clicks, leveraging SciPhi's secure and scalable infrastructure.

4. **Interact with Your Pipeline**: Use the [R2R client](/integrations/interacting-with-a-deployed-pipeline) to seamlessly interact with your deployed RAG pipeline, performing operations such as upserting entries, searching, and performing RAG completions.
