## Quick Install with `pip`:

Install R2R swiftly using `pip` to get started with minimal setup. This method will get you setup with the default configuration:

```bash
# `'r2r[all]'` to download all optional deps
pip install 'r2r[eval]'

# setup default env 
export OPENAI_API_KEY=sk-...
# ... see below for more configuration info
```


### Setup Environment

Configure environment variables for your setup:

#### Language Model providers

**OpenAI API Key**

Used by default for embedding and language models **-**

```bash
export OPENAI_API_KEY=sk-...
```

**Anthropic / Vertex AI / ...**

For optional use at runtime by embedding and language models **-**

```bash
export ANTHROPIC_API_KEY=...
export VERTEX_API_KEY=...
...
```



### Customization and Configuration

R2R provides extensive options to customize and configure your pipeline to suit specific needs. You can configure various providers as through the `config.json` file and set environment variables as shown above to specify your vector database, embedding, language model, evaluation, and logging providers.
